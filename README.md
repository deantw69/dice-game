# 🎲 骰盅遊戲(線上多人)

網頁版多人搖骰遊戲。玩家連線後取暱稱、用 4 碼房號(可自訂)加入同一房間一起玩。所有骰子點數與回合狀態都在伺服器產生與保管(server-authoritative),暗骰只有自己看得到。

## 遊戲模式

### 純搖骰
每人各搖 N 顆骰子,點數公開,依總和顯示排行,無勝負。

### 吹牛骰
每人各搖 N 顆暗骰(只看得到自己)。「抓」之前自己的骰子用**骰盅**呈現:蓋著待命,按住搖骰時盅會抖動,放開掀蓋亮出自己的點數(開盅後可點骰子區把盅反覆蓋回 / 打開)。全員搖完後任何人可按「抓(開盅)」,公開所有人骰子並統計各點數數量(以 popup 呈現)。開盅後由**房主選出本輪輸家**,輸家扣一顆骰子;骰子歸零則本場結束。

### 混合模式
從 5 顆暗骰開始的對局,每一輪由玩家先決定本局子玩法:

- **紅黑單雙** — 選一個條件(紅 / 黑 / 單 / 雙 / 大 / 小),符合的骰子被拿掉;骰子歸零者淘汰、本場結束。可開「自動順位」讓條件由玩家順位輪流決定。
- **吹牛** — 與吹牛骰相同的開盅統計;開盅後由房主選出本輪輸家,輸家扣一顆骰子。
- **話胚** — 全員開牌比撲克牌型,點數最小者可「重骰」或「認輸」。
  - 牌型大小(高→低):**豹子(五同) > 鐵支(四同) > 順子 > 葫蘆 > 三條 > 兩對 > 一對 > 散牌**;順子 `12345 > 23456`。
  - 點數強弱:**1 > 6 > 5 > 4 > 3 > 2**(五個 4 稱「4豹子」)。
  - 最小者每段有 **3 次重骰**(換人又換回來會補滿);可點選骰子「鎖定」保留點數(同段第一次用鎖定多扣 1 次),次數用完仍最小即輸。

混合模式可開「**由輸家決定玩法**」:下一場玩法由上一局輸家依順位決定。

## 其他功能
- **重連 / 暫離**:重新整理或斷線 30 秒內回到原座位;房主可把閒置玩家丟入「暫離觀戰區」,本人按「我回來了」下一局再加入。
- **房主控制**:選模式 / 骰子數、打亂玩家順序、踢人、轉移房主、強制重來。
- **便利操作**:自動下一場、自動骰、空白鍵搖骰(長按持續轉動、放開定格)、音效開關、輸的次數統計、手機版面適配。

## 技術
- 後端:Node.js + Express + Socket.IO(即時連線;房間/遊戲狀態存記憶體,重啟即清空)
- 前端:純 HTML / CSS / JS,CSS 3D 骰子動畫
- 防作弊:每位玩家收到各自的視圖(`viewFor`),他人未開的暗骰不會送到前端

## 本地執行
```bash
npm install
npm start          # 開啟 http://localhost:3000
npm run dev        # node --watch,存檔自動重啟
```
伺服器會讀取環境變數 `PORT`(部署平台會自動帶入),預設 3000。需要 Node 18+。
骰子動畫比較頁:`http://localhost:3000/demo.html`。

## 部署
任何支援長連線(WebSocket)的 Node 平台皆可(Render / Railway / Fly.io 等)。本專案內含 `render.yaml`(Render Blueprint,push 到 `main` 自動部署)。
- Build:`npm install`
- Start:`npm start`
